# palera1n loader (lite)

This is the loader application that will maybe soon be installed to your device after jailbreaking which lets you bootstrap the device among other things.

## Building

You can edit the project in Xcode and then run `make` to build the ipa file.

## IPA
If you can't build, get the IPA [here](https://static.palera.in/artifacts/loader/universal_lite/palera1nLoader.ipa)
